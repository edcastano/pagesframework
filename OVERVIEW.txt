   ├─ Sub-brand/Division → Node B (Showcase Logic)
   ├─ Product/SaaS → Node C (Product Logic)
   └─ Service → Node D (Service Logic)

Node B - Showcase Page:
   → Distinct brand? → Different audience? → Can post 2-3x/week?
   Results: Feature on Company Page | Use Targeting | Create Showcase Page

Node C - Product Page:
   → Standardized offering? → Feature-driven? → Monthly content?
   Results: Feature on Company Page | Create Product Page

Node D - Service Page:
   → Distinct practice? → Needs thought leadership? → Regular case studies?
   Results: Feature on Company Page | Create Service Page
